Configure Acadaman

Note

You'll need to have the Staff role before you can begin to configure Acadaman.

Before you can begin to reap the benefits of Acadaman, there are a number of configurations you need to do:

  • Global settings
  • Admission settings
  • Academic level
  • Customize Forms
  • Setup Fee Structures
  • Add & Configure Courses

Configure Global Settings

As part of setup process for Acadaman, you can configure your institution name, add institution logo and add about information for your learning institution.

To configure global settings:

  1. Sign in to Acadaman
  2. Click Configure
  3. Update Name, choose a logo to upload, and enter "About" information for your learning institution
  4. Click Save Configuration

Configure Admission Settings

A learning institution has a period when admission is Open or Closed. We also made it possible to set current session as well as academic term.

To configure admission settings:

  1. Sign in to Acadaman
  2. Click Configure > Admission
  3. Select your desired academic session.
  4. Select your desired academic semester

  5. Select your desired admission status

    An admission status can be either Open or Closed

Customize Academic Level

By customizing academic level, you can set the current stage of a student thereby helping to categorize which groups of students have access to specific courses.

To add custom academic level:

  1. Sign in to Acadaman
  2. Click Configure > Academic Level > Add Academic Level
  3. Click Save

Customize Mode of Entry

Applicable only to College and University learning institution types

Built-in mode of entry are: DE, UME

To add custom mode of entry:

  1. Sign in to Acadaman
  2. Click Configure > Mode of Entry > Add Mode of Entry
  3. Click Save

Add Custom Study Type

Applicable only to College and University learning institution types

Built-in study types for College and University learning institution types are Undergraduate, Postgraduate, Postgraduate diploma, Postgraduate masters, Medicine type grades, Certificate CIS, Diploma CIS, Postgraduate PhD, Postgraduate Mphil.

To add custom study types:

  1. Sign in to Acadaman
  2. Click Configure > Study Type > Add Study Type
  3. Click Save

Customize Acadaman Forms

All Acadaman administration forms are customizable. As part of actions that can be carried out on forms are:

  • Change form label text
  • Change field description
  • Change the ordering position of a form field
  • Change a form field visibility. A form field can either be visible or invisible
  • Add extra fields to a form

Add A Fee Item

A Fee Item is a type of Fee that is paid within a learning institution. For example, a user with Bursar role can setup custom Fee item.

To add a Fee Item:

  1. Sign in to Acadaman
  2. Click Configure > Fee Items > Add Fee Item
  3. Add title and description
  4. Check whether Scholarship is applicable to this Fee Item
  5. Click Save and Add another

Configure An Existing Programme

Available to University, College and Professional Learning Institutes

To configure an existing programme:

  1. Sign in to Acadaman
  2. Click Configure > Programme
  3. From the programme list, click Edit on the Programme you'll like to configure
  4. Configure appropriately and Click Save
